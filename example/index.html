It works!

Welcome to the goserve example page. If you see this page it means the goserve static file server is up and running.

Usage

goserve is a command line app that serves a directory of static files to localhost. By default the http.ServeFile go function will serve a index.html file if found.

      $ goserve -port "1234"

The above example will serve the content to of the current directory to localhost:1234

While goserve is running with the -verbose flag it prints request headers to the command line.

      GET localhost:8000/example/style.css [http://localhost:8000/example/] [Mozilla/5.0 (Macintosh; Intel Mac OS X 10_12_1) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/62.0.3202.94 Safari/537.36]
